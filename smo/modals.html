Mark Safe

Marking safe means..

In future, If site's content is messed up anyhow. in that case you can recover this version by clicking on 'Restore Last Safe' option in 'Menu'.

Restore Safe

Restoring Last Safe Means..

If site's content is messed up anyhow at present [Disaster]. In this case you can recover Last SAFE version.
Last SAFE version would have been marked by Editor in Past.
If you have not marked any Last Safe earlier, It will effect 'no change'.

Reset Factory Version

Resetting Factory Version Means..

If site's content is messed up anyhow at present [Disaster]. In this case you can recover Factory version.
Or You want to start all over again.
If you have not marked any Last Safe earlier, It will effect 'no change'.
